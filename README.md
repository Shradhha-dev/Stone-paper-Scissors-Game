# 🎮 Stone Paper Scissors Game

A simple and interactive Stone Paper Scissors game built using HTML, CSS, and JavaScript.  
This project demonstrates basic JavaScript logic, DOM manipulation, and event handling by allowing users to play against the computer in a browser.

---

## 📌 About the Project

The Stone Paper Scissors game is a beginner-friendly web project designed to strengthen core JavaScript concepts. The player selects a move (Stone, Paper, or Scissors), and the computer randomly generates its move. The game then determines the winner based on standard game rules.

This project helped practice game logic implementation and user interaction handling in JavaScript.

---

## 🚀 Features

- Interactive user interface
- Random computer move generation
- Instant result display
- Score tracking for player and computer
- Simple and responsive design

## ▶ How to Run the Project

1. Clone the repository

git clone https://github.com/shradhha-dev/stone-paper-scissors.git

2. Navigate to the project folder

cd stone-paper-scissors

3. Open the project

Open the file `index.html` in any web browser.

You can also run the project using **Live Server in VS Code**.
